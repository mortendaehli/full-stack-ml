Full stack ML
=========================================

.. toctree::
   :hidden:
   :maxdepth: 2

   user-guide

This is a powerful, *user-friendly* full stack ML demo created in Python and Electron.

This demo is powerful and easy to use:
::

   $ docker-compose up

Then you simply open your browser at https://localhost:8080
